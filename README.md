# EduCtrl-style Next.js Website

Multi-page Next.js website inspired by `eductrl.com`, rebuilt with the App Router and plain JavaScript.

## Pages

- `/`
- `/consultancy`
- `/coaching`
- `/cms`
- `/pricing`
- `/resources`
- `/about`
- `/contact`

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
